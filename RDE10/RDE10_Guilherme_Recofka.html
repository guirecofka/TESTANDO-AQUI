Cascading Style Sheets

Definição

Cascading Style Sheets - Folhas de Estilo em Cascata - tem por objetivo realizar a estilização de conteúdo de documentos HTML. Pode definir cores, fontes, espaçamento, alinhamentos, bordas, dentre outros.

Componentes

  • Seletores: selecione elementos HTML com base no nome do elemento ao qual o estilo será aplicado.
  • Propriedades: o aspecto específico do elemento que você quer usar como estilo.
  • Volume: a configuração aplicada ao atributo.

Formas de utilização

Inline

Os códigos CSS são definidos dentro de uma tag HTML Afetam somente a tag em que foi definida.
